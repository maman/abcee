Untuk Orang Tua

Situs ini digunakan untuk anak usia 3 - 5 tahun untuk belajar mengenal huruf. Penggunaannya cukup simpel, klik start, lalu ikuti garis putus putus dengan mouse / jari (untuk perangkat touchscreen). untuk memilih huruf, klik huruf yang diinginkan pada sisi kanan. untuk memilih warna, klik warna yang diinginkan pada sisi kiri. untuk menghapus coretan, klik tanda '+' di sisi kiri atas.

Tentang Kami

Dibuat dan dikembangkan oleh Achmad Mahardi, Bobi Triyanto, dan Monica untuk kuis besar mata kuliah Interaksi Manusia & Komputer di Prodi Sistem Informasi Universitas Ma Chung.

Informasi Teknis

Menggunakan library jQuery, JCanvas, dan Sketch.js untuk interaksi utama. Menggunakan Speak.js untuk audio playback. Font yang digunakan adalah Quicksand untuk huruf putus putus, serta Lato untuk paragraf dan teks lainnya.

Source Code dapat dilihat di GitHub
